Getting started - FiCo4OMNeT Simulation Models

You successfully installed the FiCo4OMNeT simulation models for fieldbus simulations. On this page we would like to give you some tips on how to get started.

Documentation

  • A changelog can be found in the WHATSNEW file
  • More info about the installation procedure can be found in the INSTALL file
  • The documentation can be found online, or generated using OMNeT++ by selecting the FiCo4OMNeT project and then opening "Project" -> "Generate NED Documentation..." (README). After generating you find the documentation in the doc folder

Examples

  • You find simulation examples in the "examples" folder
  • Special examples configured in the Abstract Network Definition Language (ANDL) are located in the "examples_andl" folder

Help

  • If you need help, you can contact us through the download page, or our webpage
  • You can find the bugtracker on the download page
